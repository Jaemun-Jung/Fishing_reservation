# 🎣 선상낚시 빈자리 텔레그램 알림

팀스카이·팀만수·지오디·금강7호·보스호·마나루·블레스·피싱아일랜드·마스터피싱·슈퍼노바·팀에프·대박
등 **여러 선사 약 30척**의 예약 현황을 감시해서, 마감된 날짜에 **빈자리가 생기면 텔레그램 단체방으로 알림**을 보냅니다.

- 로컬(내 PC): `시작하기.bat` 더블클릭
- 클라우드(24시간, PC 꺼져도): **GitHub Actions**가 5분마다 자동 실행

---

## 감시할 배·날짜 정하기

`낚시빈자리알림.ps1` 파일 위쪽 `$감시목록` 만 고치면 됩니다.

```powershell
$감시목록 = @(
    @{ 배 = "팀스카이호"; 날짜 = @("2026-08-15", "2026-08-16") }
    @{ 배 = "금강7호";    날짜 = @("2026-08-07") }
    @{ 배 = "슈퍼노바호"; 날짜 = @("2026-08-08") }
)
```

배 이름은 파일 안 "선박 목록" 주석에 있는 이름을 그대로 쓰세요.

## 내 PC에서 돌리기 (간단)

1. `설정.local.예시.txt` 를 복사해 `설정.local.txt` 로 만들고, 봇 토큰과 방번호를 채웁니다.
2. `시작하기.bat` 더블클릭 → 5분마다 자동 확인.
   - 처음 실행이 막히면 PowerShell에서 한 번만:
     `Set-ExecutionPolicy -Scope CurrentUser RemoteSigned`

## GitHub로 24시간 자동 실행하기

### 1) 봇 토큰 새로 발급 (보안)
공개 저장소에 올릴 거라, 텔레그램 **@BotFather → `/revoke`** 로 토큰을 새로 받으세요.
(코드에는 토큰이 안 들어가지만, 이전 토큰이 노출된 적 있으면 교체가 안전합니다.)

### 2) GitHub에 올리기
- GitHub 계정으로 새 저장소(public) 생성 → 이 폴더를 push
- ⚠️ `설정.local.txt` 는 `.gitignore` 로 **안 올라갑니다** (토큰 보호). 정상입니다.

### 3) Secret 등록 (토큰을 안전하게 저장)
저장소 → **Settings → Secrets and variables → Actions → New repository secret** 에서 2개 등록:
- `BOT_TOKEN` = 새로 발급한 봇 토큰
- `CHAT_ID` = 단체방 번호 (예: `-5152557020`)

### 4) Actions 켜기
- 저장소 **Actions** 탭 → 워크플로우 활성화
- `빈자리 확인` 워크플로우에서 **Run workflow** 로 즉시 테스트 가능
- 이후 5분마다 자동 실행됩니다.

> 참고: GitHub 예약 실행은 가끔 몇 분 늦을 수 있고, 저장소가 60일간 활동이 없으면 자동 실행이 멈춥니다(가끔 커밋하거나 수동 실행하면 유지).

## 구조 (개발자용)
- `낚시빈자리알림.ps1` — 1회 실행: 지정 배·날짜 조회 → 이전 상태(`상태.json`)와 비교 → 빈자리 전환 시 텔레그램 발송 → 상태 저장
- 시스템 3종 어댑터: `Get-TFAjax`(더피싱 본사) / `Get-TFGet`(더피싱 자체도메인) / `Get-SS24`(선상24)
- `설정.local.txt` / 환경변수 `BOT_TOKEN`·`CHAT_ID` 에서 비밀값 로딩
- `.github/workflows/check.yml` — 5분 cron + 상태 커밋
